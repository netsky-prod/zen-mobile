# Fonts

Download these fonts from Google Fonts:

1. **Bebas Neue** - https://fonts.google.com/specimen/Bebas+Neue
   - Download and save as `BebasNeue-Regular.ttf`

2. **Special Elite** - https://fonts.google.com/specimen/Special+Elite
   - Download and save as `SpecialElite-Regular.ttf`

Place the .ttf files in this directory.
